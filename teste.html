Teste de GIT e GITHUB

Como fazer commit e enviar ao github?

0. para clonar repositório git clone 'link do repositório'.

1. para listar as configurações do GIT: git config --global --list.

2. se não for seu usuário de email: configurar usando git config --global user.name "Seu Nome" configurar git config --global user.email "seu.email@escola.pr.gov.br

3. não esquecer que na 1ª vez irá pedir autorização do github.

4. Salvar o arquivo que está escrevendo

5. Comando para criar o arquivo: git add nomeDoArquivo.

6. Comando para gravar COMMIT: git commit nomeDoArquivo -m "sua mensagem".

7. Comando para enviar ao repositório GitHub: git push origin main.

8. Para limpar as credenciais: cmdkey /delete git:https://github.com
